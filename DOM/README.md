# DOM (Document Object Model)

This folder explains how JavaScript talks to HTML (The Application Programming Interface for Web Documents).
Simple Explanation:

- **DOM**: It creates a tree of all elements (checking, changing, deleting).
- **Selector**: How to find an element (ById, ByClassName, QuerySelector).
- **Event Listener**: Waiting for user action (Click, Type, MouseOver).
- **Manipulation**: Changing text, style, or adding new elements.

**Files:**
- `index.html`: The structure.
- `script.js`: The logic to change structure.
- `style.css`: The look.
