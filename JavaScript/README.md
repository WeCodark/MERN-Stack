# JavaScript Basics

This folder contains all the basic codes for learning JavaScript.
Simple explanation for each topic:

- **Advance.js**: Little bit complex topics like Async/Await, Promises.
- **Object.js**: How to store data in key-value pairs (properties).
- **arrays.js**: List of items stored in one variable.
- **conditionalStatement.js**: Making decisions (If-Else).
- **controlStatement.js**: Controlling flow of code.
- **dataType.js**: Different types of data like Number, String, Boolean.
- **function.js**: Reusable blocks of code.
- **loops.js**: Doing same thing again and again.
- **script.js**: General practice code.

Everything is written in simple way for easy understanding.
