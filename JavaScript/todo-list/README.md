# Project: Todo List

This folder contains the main project of this course.
Simple Explanation:

- **Todo-CrudAPI**: A Todo List App that saves data on a server (cloud).
- **Index.html**: The page design.
- **Script.js**: The logic to fetch, add, delete tasks using API.
- **API**: Like a waiter, taking order (Task) to kitchen (Server) and bringing food (Data) back.
