Before diving into MERN, I have mastered the basics:

- **[JavaScript](./JavaScript)**: The brain (Logic, Loops, Functions, Promises).
- **[DOM](./DOM)**: Connecting HTML with JS (Event Listeners, Manipulation).
- **[OOP](./OOP)**: Object Oriented Programming (Classes, Objects).
- **[Project](./Project)**: A real-world Todo App using an API.
