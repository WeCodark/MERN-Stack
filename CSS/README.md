# CSS (Cascading Style Sheets)

This folder contains styles to make website look beautiful.
Simple Explanation:

- **Basic**: Colors, Fonts, Backgrounds.
- **BoxModel**: Margin (Outside space), Padding (Inside space), Border.
- **FlexBox**: Arranging items in row or column easily.
- **GridBox**: Arranging items in rows and columns (Like table).
- **Tailwind**: A special tool to write CSS faster using classes.

CSS is like clothes and makeup for the website.
