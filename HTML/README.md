# HTML (HyperText Markup Language)

This folder covers the skeleton of a web page.
Simple Explanation:

- **Tags**: Building blocks like `<h1>`, `<p>`, `<div>`.
- **Attributes**: Extra info like `id`, `class`, `src`, `href`.
- **Forms**: Collecting user data (Inputs, Buttons).
- **Table**: Organizing data in rows and columns.
- **Semantic HTML**: Using meaningful tags (`<header>`, `<footer>`) for better understanding.
